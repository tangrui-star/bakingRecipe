# 订单数据处理功能需求文档

## 1. 功能概述

本功能用于处理团购订单的 Excel 数据，支持按跟团号筛选订单、汇总商品数量、计算配方材料用量、提取备注信息及生成打印单号，最终将所有结果导出为 Excel 文件。

**整体流程：**

```
输入跟团号 → 上传 Excel 文件 → 数据处理生成 → 多维度结果展示 → 导出 Excel
```

---

## 2. 页面结构

页面分为两个区域：

- **数据处理区**：跟团号录入 + 文件上传 + 操作按钮
- **结果输出区**：多 Tab 展示处理结果 + 导出按钮

---

## 3. 功能需求

### 3.1 跟团号录入

**描述：** 用户手动输入需要处理的跟团号，支持批量录入。

**需求细节：**

- 点击输入框后显示文本输入框，失焦或按回车触发确认
- 输入内容以空格分隔，自动解析为多个跟团号
- 只保留纯数字的跟团号，非数字内容自动过滤
- 跟团号转为数字类型后去重，并按升序排列
- 已录入的跟团号以 Tag 形式展示，每个 Tag 可单独删除
- 页面实时显示当前已录入的跟团号总数
- 点击"重置"按钮，清空所有已录入跟团号、文件及处理结果

**字段说明：**

| 字段 | 类型 | 说明 |
|---|---|---|
| dynamicTags | Array\<Number\> | 已录入的跟团号列表 |
| numberOrders | Number | 当前跟团号总数 |

---

### 3.2 Excel 文件上传

**描述：** 用户选取本地 Excel 文件作为数据源。

**需求细节：**

- 使用文件选择控件，限制只能选取 1 个文件，不自动上传
- 文件选取后暂存到本地变量，不发送到服务器
- 点击"数据处理生成"按钮时，使用 `FileReader` 读取文件内容
- 使用 `xlsx` 库解析文件（`XLSX.read`），取第一个 Sheet 的数据转为 JSON 数组
- **密码保护处理：** 若文件受密码保护，弹出密码输入框，用户输入密码后重新解析；密码错误则提示错误信息

---

### 3.3 数据处理生成

点击"数据处理生成"按钮后，依次执行以下 5 个处理步骤，所有步骤基于同一份过滤后的数据源。

#### 3.3.1 单号数据过滤

**描述：** 从 Excel 全量数据中，筛选出跟团号在已录入列表中的行。

**逻辑：**

```
遍历 Excel 所有行
  若该行的"跟团号"字段（转为 Number）存在于 dynamicTags 中
    → 保留该行
```

**输出：** 过滤后的订单行数组，作为后续所有处理的数据源。

**Excel 必需字段：** `跟团号`、`下单人`、`商品`、`分类`、`数量`、`收货人`、`联系电话`、`详细地址`、`团员备注`、`团长备注`、`支付时间`、`订单金额`、`退款金额`、`订单状态`

---

#### 3.3.2 商品数量汇总

**描述：** 按商品和分类聚合数量，并记录每个商品来自哪些跟团号。

**逻辑：**

```
以"商品 - 分类"为 key，遍历过滤后数据：
  累加该商品的数量
  拼接跟团号及数量信息，格式：、"跟团号:数量"

排序规则：
  1. 按分类字母序升序
  2. 同分类内按数量降序

重新分配序号 id（从 1 开始）
```

**输出字段：**

| 字段 | 说明 |
|---|---|
| id | 序号 |
| 分类 | 商品分类 |
| 商品 | 商品名称 |
| 数量 | 该商品总数量 |
| 跟团号及数量 | 各跟团号对应数量明细，如 `->、"101:2"、"102:1"` |

---

#### 3.3.3 配方计算

**描述：** 基于商品数量汇总结果，按预设配方规则计算各材料的实际用量。

**逻辑：**

- 系统内置 21 条配方规则（A1 ~ A21），每条规则对应一种或多种商品
- 每条规则从汇总数据中查找对应商品数量，按规则计算配方批次数（`formulaNumber`），再乘以各材料基础单位重量得出实际用量
- 若对应商品数量为 0，则该配方不计算（保持初始值）

**配方批次数计算方式（各规则不同）：**

| 计算方式 | 说明 |
|---|---|
| `Math.ceil(数量 / N)` | 向上取整除以 N |
| `Math.floor(数量 / N)` | 向下取整除以 N |
| `Math.round(数量 / N)` | 四舍五入除以 N |
| `Math.ceil(数量 / N) + 1` | 向上取整后加 1 |
| `(数量 + 1) / 2` | 加 1 后除以 2 |
| `数量A + 数量B` | 多商品数量合并 |

**配方规则列表（A1 ~ A21）：**

| 配方ID | 配方名称 | 关联商品 | 批次数规则 | 材料（最多7项） |
|---|---|---|---|---|
| A1 | 番茄肉松贝果 | 芝士番茄全麦肉松贝果(容易炸口)(生重140克) | `ceil(数量/3)` | 肉松75g、芝士30g、番茄15g、辣椒5g |
| A2 | 咸蛋黄 | 全麦咸蛋黄肉松碱水(生重150克) + 艾草咸蛋黄肉松贝果(生重135克左右) | 两商品数量之和 | 蛋`floor(总数/4)`个、肉松35g |
| A3 | 章鱼烧 | 章鱼烧贝果(生重150克左右) | 商品数量 | 肠`round(数量/4)`个、肉松30g、海苔5g、酱2.5g |
| A4 | 鸡枞菌 | 低油无糖鸡枞菌恰巴塔(生重180克左右) | `ceil(数量/3)+1` | 面300g、水250g、油20g、鸡枞菌50g、酵母粉1g、盐4g |
| A5 | 鸡肉欧包 | 无油糖葱香鸡肉干乳酪丁欧包(生重500克左右) | `(数量+1)/2` | 面500g、洋葱10g、叶子2g、水320g、酵母粉2g、盐10g |
| A6 | 鸡肉欧包后加 | 无油糖葱香鸡肉干乳酪丁欧包(生重500克左右) | `(数量+1)/2` | 鸡肉50g、乳酪丁20g |
| A7 | 青提欧包 | 无油低糖酒渍青提茉莉欧包(生重500克左右) | `(数量+1)/2` | 面500g、酱50g、茉莉粉4g、抹茶粉2g、水290g、酵母粉2g、糖10g |
| A8 | 青提欧包后加 | 无油低糖酒渍青提茉莉欧包(生重500克左右) | `(数量+1)/2` | 葡萄干50g、花10g |
| A9 | 曲奇 | 美式大曲奇（全分类）- 特定商品 + 燕麦曲奇/2 | `ceil(综合计算结果/6)` | 鸡蛋`(批次数-1)`个、油125g、面215g、糖50g、泡打粉1g |
| A10 | 燕麦片 | 每日可可黑巧坚果多多烤燕麦片小包(50克左右) + 无油黑巧可可坚果烤燕麦片(主重180克左右) | 两商品数量之和 | 燕麦片小包=小包数量、燕麦片大包=大包数量 |
| A11 | 煮红豆 | 古早红豆肉松碱水(生重150克左右) + 抹茶红豆碱水改良版(生重140克) | 两商品数量之和 | 红豆`总数×15`g |
| A13 | 黑芝麻粉 | 黑芝麻花生双拼贝果(生重150克) | `ceil(数量/2)` | 黑芝麻粉40g、奶粉10g、糖2g |
| A14 | 花生（白） | 无油糖肉桂花生碱水(生重150克左右) + 全麦可可巧克力奇亚籽花生贝果(生重160克左右) | 两商品数量之和 | 肉桂花生`50×A数量`g、肉桂粉`3×A数量`g、糖`2×A数量`g、可可花生`50×B数量`g、可可粉`2×B数量`g、糖`2×B数量`g |
| A15 | 花生（白） | 黑芝麻花生双拼贝果(生重150克) + 香蕉布朗尼花生贝果(生重150克) | 两商品数量之和 | 黑芝麻花生`25×A数量`g、香蕉花生`50×B数量`g、糖`合计花生/25`g |
| A16 | 花生（粗） | 全麦花生贝果(生重150克) + 海盐花生碱水包(生重150克) | 两商品数量之和 | 花生`50×总数 + 花生奇亚籽黑巧美式曲奇数量×30`g |
| A17 | 花生（细） | 全麦花生坚果辣松贝果(生重150克) + 全麦花生坚果辣松碱水(生重150克) + 花生酱(100克一瓶) | 三商品数量之和 | 花生`30×(A+B数量) + 100×C数量`g、肉松`20×(A+B数量)`g、花生酱`C数量`瓶 |
| A18 | 腰果 | 无油糖椰香斑斓腰果贝果(生重150克左右) + 无油糖抹茶腰果碱水(生重150克左右) | 两商品数量之和 | 腰果`50×总数 + (斑斓酱+抹茶腰果酱)×100`g |
| A19 | 斑斓腰果贝果 | 无油糖椰香斑斓腰果贝果(生重150克左右) | 商品数量 | 斑斓50g、斑斓粉2g、糖1g、椰奶粉2g |
| A20 | 抹茶腰果碱水 | 无油糖抹茶腰果碱水(生重150克左右) | 商品数量 | 抹茶50g、抹茶粉2g、糖1g |
| A21 | 斑斓酱 | 斑斓椰香奇亚籽腰果酱(100克左右) + 抹茶腰果酱(100克一瓶) | 两商品数量之和 | 斑斓酱=A数量瓶、抹茶腰果酱=B数量瓶 |

**输出字段：**

| 字段 | 说明 |
|---|---|
| formulaNumber | 配方批次数 |
| formulaName | 配方名称 |
| subitem1 ~ subitem7 | 材料名称（最多7项） |
| subitem1Weight ~ subitem7Weight | 对应材料实际用量 |

---

#### 3.3.4 团员备注提取

**描述：** 提取有团员备注的订单行。

**逻辑：**

```
遍历过滤后数据：
  若"团员备注"字段不为空且不为 null
  且该跟团号尚未出现过（按跟团号去重）
    → 保留该行
```

**展示字段：** 跟团号、下单人、团员备注、团长备注、商品、分类、订单金额、退款金额、订单状态、联系电话

---

#### 3.3.5 团长备注提取

**描述：** 提取有团长备注的订单行，逻辑同团员备注，字段改为"团长备注"。

---

#### 3.3.6 批量打印单号生成

**描述：** 为每个跟团号生成格式化的打印内容。

**逻辑：**

```
遍历过滤后数据，按跟团号去重：
  生成打印内容格式：
    第1行：跟团号
    第2行：收货信息：{收货人} {联系电话}
    第3行起：详细地址（每16个字符换行）
```

**输出字段：**

| 字段 | 说明 |
|---|---|
| formulaNumber | 跟团号 |
| formulaInfor | 格式化后的完整打印内容 |

---

### 3.4 结果展示

**描述：** 处理完成后，通过 Tab 页分组展示所有结果，每个 Tab 使用分页表格展示，每页默认 20 条。

| Tab 名称 | 数据来源 | 主要展示字段 |
|---|---|---|
| 数量汇总 | 商品数量汇总结果 | 序号、分类、商品、数量、跟团号及数量 |
| 团员备注 | 团员备注提取结果 | 跟团号、下单人、团员备注、团长备注、商品、分类、订单金额、退款金额、订单状态、联系电话 |
| 团长备注 | 团长备注提取结果 | 同上 |
| 单号数据源 | 单号过滤结果 | 跟团号、下单人、团员备注、支付时间、团长备注、商品、分类、数量、订单金额、退款金额、订单状态、收货人、联系电话、详细地址 |
| 配方计算 | 配方计算结果 | 配方数量、配方名称、材料项1~7及对应重量 |
| 批量打印单号 | 打印单号生成结果 | 跟团号、基础信息（格式化打印内容） |

---

### 3.5 导出 Excel

**描述：** 将所有处理结果导出为一个多 Sheet 的 Excel 文件。

**需求细节：**

- 点击"Download"按钮触发导出
- 使用 `xlsx` 库创建工作簿（`XLSX.utils.book_new()`）
- 将 6 个结果分别写入独立 Sheet，Sheet 名称格式为 `M月D日{内容名称}`
- 数量汇总 Sheet 对同分类的行做合并单元格处理（第一列按分类合并）
- 配方计算 Sheet 导出字段：配方名称、材料项1~7及对应重量（不导出配方ID和备注）
- 批量打印单号 Sheet 只导出"跟团号"字段（即格式化后的完整打印内容）
- 文件名格式：`M月D日数据结果.xlsx`
- 通过动态创建 `<a>` 标签并触发 `click` 事件实现浏览器下载

**导出 Sheet 列表：**

| Sheet 名称 | 数据来源 |
|---|---|
| M月D日数据汇总 | 商品数量汇总 |
| M月D日团员备注 | 团员备注 |
| M月D日团长备注 | 团长备注 |
| M月D日单号数据源 | 单号过滤结果 |
| M月D日配方计算 | 配方计算结果 |
| M月D日批量打印单号 | 打印单号 |

## 4. 数据流向

```
用户输入跟团号列表
        ↓
用户选取 Excel 文件
        ↓
点击"数据处理生成"
        ↓
FileReader 读取文件 → xlsx 解析为 JSON
        ↓
① 单号数据过滤（按跟团号筛选）
        ↓
┌───────────────────────────────────┐
│  ② 商品数量汇总                    │
│  ③ 配方计算（依赖②的结果）          │
│  ④ 团员备注提取                    │
│  ⑤ 团长备注提取                    │
│  ⑥ 批量打印单号生成                 │
└───────────────────────────────────┘
        ↓
Tab 页展示各结果
        ↓
点击"Download"导出多 Sheet Excel
```

---

## 5. 异常处理

| 场景 | 处理方式 |
|---|---|
| 文件受密码保护 | 弹出密码输入框，带密码重新解析 |
| 密码错误 | 弹出提示"密码错误，请重新上传文件并输入正确密码" |
| 文件格式错误 | 弹出提示"读取文件时出错，请检查文件格式" |
| 未找到工作表 | 弹出提示"未找到工作表，请检查文件内容" |
| 未上传文件直接处理 | 不做处理（依赖 fileData 为空时不执行） |
